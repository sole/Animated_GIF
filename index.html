Animated_GIF

A library that makes creating animated GIFs easy. Everything is client-side, using JavaScript.

Get the source and start animating today!

Here are a few examples demonstrating what the library can do --and serving as documentation.

  • Basic - demonstrates how to build both single frame and multi frame GIFs
  • Custom Palette - shows how to use custom palettes
  • Dithering - demonstrates the different types of dithering that can be used
  • Sample Interval - demonstrates the effect of the sampleInterval option
  • Stress - this is a test to see how well browsers work when creating many GIFs quickly
